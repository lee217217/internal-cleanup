部署設定

  1. 上傳整個專案到 Netlify。
  2. 只需在 Netlify env 設定 PPLX_API_KEY。
  3. Base directory 留空，publish 目錄為 public。
  4. 不用再設 IMAGE_EDIT_API_URL 與 IMAGE_EDIT_API_KEY。
這版的限制

真正修補是本地遮罩模糊法，較適合小範圍污點、遮擋、簡單標記。Perplexity 公開文件目前重點是圖像附件分析，不是穩定公開的像素級修圖 API。[web:62][web:65]
